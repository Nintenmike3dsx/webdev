theHunter: Call of the Wild Let's Play and Review

Introduction:

theHunter: Call of the Wild has the description Discover an atmospheric hunting game like no other in this realistic, stunning open world - regularly updated in collaboration with the community. Immerse yourself in the single player campaign, or share the ultimate hunting experience with friends. theHunter is a fun first person hunting simulation game. This game was released on Feburary 16th, 2017.

DEVELOPER: Expansive Worlds
PUBLISHER: Avalanche Studios.

Steam Review:

        "Very fun game, I like to play it on multiplayer with my family and we compare trophies and the like. After 220 hours of playing these are my thoughts. 
        The base game is very fun, but the DLC's are where the real meat of the game is (pun intended). Some of the DLC's like the atv and blinds/stands should 
        be included in the base game, because the game is very hard to play without them. I would have a hard time playing the game without tents, blinds, scopes, 
        rifles, and other DLC items that seem to be fundamental to the gameplay. I understand that this is the way that the game makes more money, but that should 
        be done through DLC maps and rare items, not things that everyone needs. However, this game is so replayable. Every time I hop on, I find new need zones and 
        parts of maps that I've never seen before. Currently, I've started a white tail deer grind on Laton Lakes. This is so much fun and I have a great time 
        hunting with a compound bow and deer callers. I would absolutely recommend this game because it has great replayability with new rifles and weapons to 
        try out, new sections of maps to discover, very fun multiplayer w/ challenges, and beautiful scenery."
         DeezAstronauts

Gameplay:

theHunter: Call of the Wild has a total of 18 reservations to play on. Two reservations come with the base game Hirschfelden Hunting Reserve and Layton Lake District.

Gameplay:

theHunter uses a mouse and keypad to control a player in a first person view. The character is able to use UI to select their invintory of weapons, callers, blinds, and more to help them hunt the veriety of animals in the reserves.
